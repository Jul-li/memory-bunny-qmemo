# 更新日志

本文件记录记忆兔 QMemo 原生 iOS 主线的重要功能与结构变化。

## 2026-06-22

### 新增

- 将原 `分类` 一级页面调整为 `统计`，新增月份日历、记录日期标记、月度汇总、本周趋势和分类统计。
- 日历支持按钮及左右拖动切换月份，拖动时相邻月份随手势进入视图。
- 本周趋势支持在当前月份内切换周次，并显示当前为第几周；到达第一周或最后一周时禁用对应方向按钮。
- 新增设置一级页面，并保持原有三栏底部导航结构。
- 新增 `AGENTS.md`、`CONTRIBUTING.md` 和 `docs/coding-rules.md`，补充项目协作、编码和验证规则。

### 调整

- 将原生 iOS 代码按 `App`、`Core`、`Features` 和 `Resources` 重新组织，并同步更新 Xcode 工程引用和 Info.plist 路径。
- 将月度统计压缩为单行四项横向布局，缩小图标、字体和卡片高度，不提供点击或横向滑动交互。
- 将统计页顶部标题固定，日历及其下方内容独立滚动，标题与滚动区域保持 6pt 间距。
- 更新 README、开发路线图、代码风格和原生开发状态文档，使其与当前工程结构和功能一致。

### 验证

- 使用 iPhone 17 / iOS 26.5 模拟器完成统计页月份切换、日期选择、周次边界、固定标题滚动和既有交互回归。
- 使用 Debug 配置完成原生 iOS Simulator 构建。
