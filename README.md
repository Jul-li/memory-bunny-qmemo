# 记忆兔 QMemo

## 1. 项目背景和目标

记忆兔是一款面向年轻用户的 Q 版卡通风本地备忘录 App。目标是让用户无需登录注册，就能快速记录生活灵感、待办事项、学习笔记和日常碎片，并通过柔和配色、圆润卡片、可爱图标和轻量交互，营造像随身手账一样的记录体验。

## 2. 已确定的关键决策和方向

- 技术栈：React Native + Expo + TypeScript，使用 Expo Router 做页面导航。
- 数据方向：无后端、无登录，备忘录数据保存到手机本地，当前通过 `MemoContext` 统一管理。
- 产品结构：首页、编辑页、分类页、设置页已经建立。
- 视觉方向：Q 版卡通、奶油黄背景、浅粉/薄荷绿/天空蓝等柔和色，卡片大圆角和轻阴影。
- 资源方向：使用 `Q版icon` 文件夹中的头像、分类、贴纸和底部导航 PNG 资源。
- 交互方向：首页支持搜索、分类筛选、卡片列表、底部导航和新建入口展开动画。
- 规范方向：`docs/CODE_STYLE_GUIDE.md` 是后续样式、字体、间距和交互修改的统一规范来源。

## 3. 未解决问题和待讨论点

- 新建展开面板的动效可继续微调节奏、锚点和内容密度。
